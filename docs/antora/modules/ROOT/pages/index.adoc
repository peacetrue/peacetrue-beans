== 简介（What）

{website}/{app-name}/[{app-name}] 是一个封装常用 Bean 属性接口的资源库。
共提供 23 个常用属性，每个属性涉及 3 个接口：一个读接口、一个写接口和一个读写接口，
例如：`Id`、`IdCapable`、`IdAware`。

== 背景（Why）

Bean 是由属性构成的，属性是 Bean 的基本元素，
抽象出属性接口可提供最细粒度的复用。

== 用法（How）
